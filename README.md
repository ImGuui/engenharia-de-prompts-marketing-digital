# 📘 Engenharia de Prompts aplicada ao Marketing Digital e Criação de Ofertas

## 🧠 Contexto e Objetivos

Este projeto tem como objetivo explorar o uso da Inteligência Artificial como ferramenta de **aprendizagem ativa e aplicação prática no marketing**, com foco na criação de ofertas e campanhas promocionais.

O estudo foi desenvolvido utilizando o NotebookLM para organizar conteúdos, testar prompts e estruturar conhecimento de forma eficiente.

### 🎯 Objetivos:
- Aprender a criar prompts mais eficientes para marketing  
- Utilizar IA para gerar **ofertas, anúncios e conteúdos promocionais**  
- Melhorar a qualidade de campanhas com base em testes e ajustes  
- Desenvolver um material reutilizável para uso profissional

## 🤖 Engenharia de Prompts e Aprendizados

### 🔍 Prompt 1 (Simples)
**Pergunta:**  
> Crie uma legenda para promoção de carne.

**Resultado:**  
Resposta genérica, pouco atrativa.

**Aprendizado:**  
Prompts vagos geram respostas fracas.

---

### 🔍 Prompt 2 (Refinado)
**Pergunta:**  
> Crie uma legenda para promoção de carne bovina para supermercado, com tom urgente, foco em preço baixo e CTA para atrair clientes.

**Resultado:**  
Texto mais chamativo e direcionado para vendas.

**Aprendizado:**  
Adicionar contexto + objetivo melhora drasticamente o resultado.

---

### 🔍 Prompt 3 (Profissional)
**Pergunta:**  
> Crie 3 variações de legenda para promoção de carne bovina em supermercado, com foco em urgência, economia e gatilhos mentais. Inclua emojis e CTA.

**Resultado:**  
Conteúdo pronto para uso em redes sociais.

---

### 🔍 Prompt 4 (Aplicação real - campanha)
**Pergunta:**  
> Crie uma campanha promocional para "Quarta Verde" com foco em hortifruti, incluindo slogan, ideias de posts e texto para anúncio.

**Resultado:**  
Estrutura completa de campanha.

---

### ⚠️ Dificuldades ("Cicatrizes")
- Respostas muito genéricas no início  
- Necessidade de detalhar melhor o objetivo  
- Ajustes constantes no tom (mais agressivo para vendas)  

💡 **Insight principal:**  
Quanto mais específico o prompt, melhor o resultado.

## 📖 Miniguia de Estudo

### 🧾 Resumo

- Engenharia de prompts é a prática de **estruturar perguntas para obter melhores respostas da IA**  
- Prompts eficazes incluem:
  - Contexto  
  - Objetivo  
  - Público-alvo  
  - Formato da resposta  
- No marketing, isso permite criar:
  - Legendas  
  - Ofertas  
  - Anúncios  
  - Campanhas completas  

---

### 📚 Glossário

- **Prompt:** instrução dada para a IA  
- **CTA (Call to Action):** chamada para ação (ex: “compre agora”)  
- **Copywriting:** escrita persuasiva para vendas  
- **Gatilhos mentais:** técnicas para influenciar decisões (urgência, escassez, etc.)  

---

### 🤖 Prompts Reutilizáveis

📌 Para ofertas:
- "Crie uma legenda promocional para [produto] com foco em preço baixo e urgência"

📌 Para campanhas:
- "Crie uma campanha completa para [tema], incluindo slogan, posts e anúncios"

📌 Para redes sociais:
- "Crie 3 variações de legenda com emojis e CTA para [produto]"

📌 Para melhoria:
- "Melhore esse texto deixando mais persuasivo: [cole aqui]"

📌 Para ideias:
- "Sugira ideias criativas de promoção para supermercado"
